The Cooking Project is a **software system designed to manage cooking recipes or food-related processes**. It focuses on organizing ingredients, cooking steps, and user interactions in a structured way. The project was developed collaboratively by Ahmad and Yazeed as part of an academic assignment.

The system demonstrates teamwork, logical structuring, and basic software engineering principles.

### README

#### Overview
A cooking-related application that helps users manage recipes, ingredients, or cooking steps efficiently.

#### Features
- Recipe management
- Ingredient listing
- Step-by-step cooking process
- User-friendly design

#### Technologies Used
- Java / Python (depending on implementation)
- Basic UI / Console interaction

#### How to Run
1. Open the project in an IDE
2. Run the main file

#### Purpose
Academic project focusing on logic building and collaborative development.
